# Speech to Text

A browser-based speech-to-text app with live transcription and MP3 export.

## Features

- Live transcript while you speak (English, Hindi, Marathi)
- Continuous recording until you click **Stop**
- Full session saved as MP3 after stopping
- Language selection via BCP-47 tags (`en-IN`, `hi-IN`, `mr-IN`)

## Requirements

- Chrome or Edge (Web Speech API)
- Microphone access

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Build

```bash
npm run build
npm run preview
```
